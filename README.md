# PROYECTO RECKITT
## Definir la Base de Datos del Proyecto de empresa aliada

Reckitt México, es una empresa líder en el mercado en productos de higiene del hogar, 
salud y nutrición, con marcas como Vanish, Lysol, Tempra, Sico, Durex y Chocomilk, entre 
muchas otras. 
Hoy en día, una de las industrias con mayores aplicaciones de analítica de datos es la 
industria de Retail o Consumo Masivo. En este sector es común analizar grandes 
volúmenes de información para anticipar tendencias y desarrollar propuestas 
innovadoras que fortalezcan la venta al consumidor final. Todo esto permite 
diseñar mejores estrategias de mercadotecnia, presencia, precios y distribución, 
ayudando a incrementar las ventas frente a la competencia.
